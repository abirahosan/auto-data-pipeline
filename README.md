# Auto Data Pipeline Tool

A simple web app to upload CSV files, automatically process the data, and download cleaned results.

## Features
- Upload CSV files
- Handle missing values, outliers, normalization, and feature selection
- Download processed CSV
- Easy-to-use web interface
